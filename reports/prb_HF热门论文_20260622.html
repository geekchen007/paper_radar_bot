核心贡献：

关键结果：

  • 参数量仅为 FLUX.1-Fill-Dev 的不到 2%（0.22B vs 11.9B）。

核心贡献：

  • 提出了DragMesh-2框架，实现了从以物体为中心的运动生成到基于物理接触驱动的灵巧手-铰接物体交互的跨越。
  • 提出了PICA（物理信息接触感知）训练机制，在无需触觉或力反馈的情况下将物理信号注入策略学习，显著提升了策略对接触负载变化的鲁棒性。
  • 构建并提供了一个纯几何的灵巧交互资源库，以支持未来的移动操作（loco-manipulation）和人形机器人手物交互研究。

关键结果：

  • 在七个GAPartNet物体类别的系统评估中，DragMesh-2在多种阻尼条件下均保持了高任务成功率。
  • 相比于现有基线方法，DragMesh-2在面对接触负载（如阻尼）变化时展现出显著更强的鲁棒性。

阅读建议：

本文适合从事机器人灵巧操作、手物交互（HOI）以及强化学习策略训练的研究人员阅读。重点关注PICA机制如何巧妙地在无触觉传感器的情况下隐式引入物理接触信号，这对于降低硬件成本和提升Sim2Real迁移能力具有重要参考价值。

核心贡献：

  • 提出了 Multi-LCB 基准，将 LiveCodeBench 的评估范围从单一的 Python 扩展到 12 种编程语言。

  • 对 24 个主流 LLM 进行了全面的多语言代码生成评估，揭示了当前模型中存在的 Python 过拟合和特定语言污染问题。

关键结果：

  • 评估结果显示，当前 LLM 在不同编程语言之间的代码生成性能存在巨大差异。
  • 发现了模型对 Python 存在明显的过拟合现象，以及特定语言的训练数据污染证据。

  • 设计了包含场景记忆（Scene Memory）和代理记忆（Agent Memory）的时间记忆机制，有效整合了跨帧和跨推理步骤的上下文信息。
  • 构建了包含30万条空间轨迹的数据集 S-300K，并通过监督微调（SFT）训练出紧凑高效的 S-Agent-8B 模型，其性能可媲美最先进的闭源大模型。

关键结果：

  • 在免训练（training-free）设置下，S-Agent 在多视角和视频空间推理基准上持续提升了各类开源和闭源 VLM 的性能。
  • 微调后的 S-Agent-8B 模型在空间推理任务上显著超越了同等规模的基线模型（如 Qwen3-VL-8B）。
  • S-Agent-8B 的表现达到了与先进闭源模型（如 GPT-5.4 和 Gemini 3）相当的水平。
